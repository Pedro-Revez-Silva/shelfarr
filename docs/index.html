Beautiful Book Discovery

Browse millions of books through Open Library and Hardcover. See high-res covers, detailed descriptions, series numbers, and edition details at a glance.

  • Instant Search: Query by title, author, or ISBN with auto-suggestions
  • Rich Metadata: Full series order, narrators, publishers, and release dates
  • Format Toggles: Seamlessly switch between audiobook and ebook editions
  • One-Click Request: Automated fulfillment or manual file uploads
shelfarr.local/search?q=harry+potter

Automated Acquisition

Let Shelfarr handle the heavy lifting. It queries your indexers, scores results against your custom format preferences, and manages downloads automatically.

  • Indexer Search: Seamless Prowlarr, Jackett & NZBHydra2 integration
  • Direct Sources: Direct ebook & audiobook downloads without torrent clients
  • Download Clients: qBittorrent, Decypharr, SABnzbd, NZBGet, Deluge & Transmission
  • Smart Priority Routing: Auto-selection rules per indexer and download client
shelfarr.local/acquisition/engine

Everything You Need

Multi-User & Roles

Users request books, admins approve and manage. Role-based access keeps things organized.

Two-Factor Auth

Secure your instance with TOTP-based 2FA and backup codes for all user accounts.

OIDC/SSO

Single sign-on via OpenID Connect. Works with Authentik, Authelia, Keycloak, and more.

Notifications

In-app, Discord, Telegram and generic webhook notifications for every request event.

Direct Downloads

Ebooks from Anna's Archive and Z-Library, plus free public-domain audiobooks from LibriVox — no torrent client needed.

Third-Party StoresBeta

See legitimate DRM-free ebook offers from supported sellers in a separate section. Checkout and payment always stay with the store.

Audible BackupBeta

Browse and preserve audiobooks you already own, confirm a safe one-time backup of eligible existing purchases, schedule hourly or longer library refreshes, and optionally back up future purchases using the Libation companion.

Manual Uploads

Upload your own ebook or audiobook files to fulfill a request when automation can't find it.

Format Preferences

Preferred, approved and rejected formats, plus bitrate and single-file rules for auto-selection.

Download Routing

Route specific indexers to specific download clients per torrent or usenet result.

Smart Organization

Path and filename templates organize files by author, series and title, ready for Audiobookshelf.

Library Sync

Automatic Audiobookshelf scans after downloads complete. Your books appear instantly.

REST API

Scoped, user-owned API tokens for search and request automation under /api/v1.

Docker Ready

Deploy in minutes with Docker. Works perfectly with your existing container setup.

Quick Start with Docker

terminal — shelfarr setup 
# Create directory and get compose file
$ mkdir shelfarr && cd shelfarr
$ curl -O https://raw.githubusercontent.com/Pedro-Revez-Silva/shelfarr/main/docker-compose.example.yml
$ mv docker-compose.example.yml docker-compose.yml

# Edit paths in docker-compose.yml, then start
$ docker-compose up -d

✓ Container started · listening on :5056

# Access at http://localhost:5056 — the first user to register becomes admin
